### Getting Started

Requirements:

- Node
- Yarn

```shell
# clone
git clone git@github.com:bfreeds/express-server-example.git

cd express-server-example

# install dependencies
yarn

# start dev server in watch mode
yarn watch
```

### Deployment

This was deployed to heroku with:

```shell
heroku create

git push heroku master
```
